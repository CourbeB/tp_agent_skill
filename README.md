# TP Agent Skill

Ce dépôt contient des ressources et des implémentations pour des custom instructions pour les agents et les AgentAgent Skills.

## Vue d'ensemble

L'objectif principal de ce projet est de démontrer comment créer et empaqueter des compétences pour les agents IA. Actuellement, il inclut :

- **pdf-to-markdown** : Une compétence robuste pour extraire du texte et des tableaux à partir de fichiers PDF et les convertir en Markdown propre et structuré.

## Pour commencer

- **Tutoriel** : Consultez [TP_Agent_Skill.md](TP_Agent_Skill.md) pour le guide étape par étape du hands-on.
- **Instructions pour l'Agent** : Voir [AGENTS.md](AGENTS.md) pour des instructions personnalisées sur la manière dont les agents doivent interagir avec ce dépôt.
